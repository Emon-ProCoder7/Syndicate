# Syndicate MVP: Vercel Deployment Guide

This project is a high-performance React application built with Vite and GSAP, translating the Syndicate design system into a production-ready codebase.

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Locally**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## ☁️ Vercel Deployment

This project is pre-configured for Vercel. To deploy:

1. **GitHub Push**: Push this directory to a new GitHub repository.
2. **Vercel Import**: Go to [vercel.com](https://vercel.com) and import the repository.
3. **Framework Preset**: Vercel will automatically detect **Vite**.

## 🎨 Tech Stack
- **Framework**: React 18 + Vite (SWC)
- **Motion**: GSAP 3 + ScrollTrigger
- **Icons**: Lucide React
- **Styling**: Vanilla CSS (Industrial Dark Mode)

## 📍 Key Features
- **3D Card Interactive**: Deal cards track cursor movement with 3D depth.
- **Scrollytelling**: Match journey reveals itself as you scroll.
- **Digital Pulse**: Premium entrance animations on load.
